## Idea

A prespectrum is a sequence of pointed types `A n` equipped with pointed maps
`ε : A n →∗ Ω (A (n+1))`.

## Definition

```agda
Prespectrum : (l : Level) → UU (lsuc l)
Prespectrum l =
  Σ (ℕ → Pointed-Type l) (λ A → (n : ℕ) → A n →∗ Ω (A (succ-ℕ n)))
```
