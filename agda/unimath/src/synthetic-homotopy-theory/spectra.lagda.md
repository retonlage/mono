## Idea

A spectrum is an infinite sequence of pointed types `A` such that
`A n ≃∗ Ω (A (n+1))` for each `n : ℕ`.

## Definition

```agda
Spectrum : (l : Level) → UU (lsuc l)
Spectrum l =
  Σ (ℕ → Pointed-Type l) (λ A → (n : ℕ) → A n ≃∗ Ω (A (succ-ℕ n)))
```
